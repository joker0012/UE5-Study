# 模块 4: GAS (Gameplay Ability System)

UE5 官方技能框架。GA / GE / AS / Tag / Cue 五大组件协作，从入门到 Prediction 全覆盖。

## 子目录

| # | 目录 | 知识点 |
|---|------|--------|
| 01 | `asc-init/` | ASC 初始化 + 绑定输入 |
| 02 | `fireball-ga/` | 第一个 GameplayAbility — 火球术 |
| 03 | `attribute-set/` | AttributeSet — HP / MP / Str |
| 04 | `gameplay-effect/` | GameplayEffect — 伤害 + 持续 Buff |
| 05 | `gameplay-tag/` | GameplayTag — 状态管理 |
| 06 | `gameplay-cue/` | GameplayCue — 视觉/音效反馈 |
| 07 | `exec-calc/` | MMC / ExecCalc — 自定义伤害公式 |
| 08 | `prediction/` | Prediction — 客户端预测 |
