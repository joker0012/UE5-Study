# 模块 3: Enhanced Input 系统

UE5 新一代输入框架。从 InputAction 创建到自定义 Trigger/Modifier 全覆盖。

## 子目录

| # | 目录 | 知识点 |
|---|------|--------|
| 01 | `basics/` | InputAction + InputMappingContext 基础 |
| 02 | `modifier/` | InputModifier (Negate / DeadZone / Swizzle) |
| 03 | `trigger/` | InputTrigger (Tap / Hold / Chord / Combo) |
| 04 | `imc-switch/` | 多 IMC 切换 |
| 05 | `cpp-binding/` | UEnhancedInputComponent 绑定 |
| 06 | `custom-trigger/` | 自定义 InputModifier / Trigger |
